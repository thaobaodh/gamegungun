HƯỚNG DẪN CÀI ĐẶT VÀ BUILD CHƯƠNG TRÌNH GAME

Bước 1: Mở Unity Hub (Link tải: https://unity3d.com/get-unity/download)

Bước 2:  Cài đặt phiên bản editor mới nhất: Chọn tab Installs => Install Editor => chọn phiên bản mới nhất.

Bước 3: Chọn Tab Projects => Open => chọn đường dẫn folder game 
        Ví dụ: C:\Download\TopGunGun

Bước 4: Lựa chọn phiên bản phù hợp để mở file game

Bước 5: Hoàn thành

Sau khi hoàn thành, để chạy Game.
Trong Unity chúng ta chọn Play trong Edit hoặc nhấn tổ hợp phím Ctrl + P để chạy game.
